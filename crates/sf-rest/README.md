# busbar-sf-rest

Salesforce REST API client (CRUD, Query, Describe, Composite, Collections).

This crate is part of the **busbar-sf-api** workspace.

- Prefer the facade crate for most usage: https://crates.io/crates/busbar-sf-api
- Docs: https://docs.rs/busbar-sf-rest
- Repo: https://github.com/composable-delivery/busbar-sf-api

## When to use this crate directly

Use `busbar-sf-rest` if you only want the REST API client without pulling in Bulk/Metadata/Tooling.
